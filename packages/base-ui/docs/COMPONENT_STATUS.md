# Component Status — @omniviewdev/base-ui

Tracks the implementation status of components in the package.

| Component | Status | Description |
|---|---|---|
| Calendar | Stable | Single-date and range-mode calendar grid; supports `min`/`max`, custom `isDateDisabled`, `weekStartsOn`, `locale`, and `autoFocus` |
| DatePicker | Stable | Text input with live parse, calendar popover; supports `min`/`max`, `format`, `locale`, `disabled`, `readOnly` |
| DateRangePicker | Stable | Text input with live range parse + calendar range popover; supports `min`/`max`, `format`, `locale`, `rangeSeparator`, `disabled`, `readOnly` |
| DateTimePicker | Stable | Text input with live parse, calendar + time-picker popover; supports `min`/`max`, `format`, `showSeconds`, `hourCycle`, `minuteStep` |
| TimePicker | Stable | Text entry + column-selector popover; supports `showSeconds`, `hourCycle`, `minuteStep`, `disabled`, `readOnly` |
